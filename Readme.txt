Project : Youtube data ingestion pipeline
Description : To identify the total time required to complete a playlist on a youtube channel.
Tech stack : Python 3.11.4
			 Libraries : Pandas , googleapiclient , sqlalchemy
Overview:
Extracting the content of all the videos in a playlist using Python by leveraging the
Google api using youtube service.
Loading the extracted information in oracle database.
Sql query to get the total time required to go through all the videos in the playlist in HH:MI:SS format
